# TheEnd 项目开发路线图

本文档旨在规划 TheEnd 项目的开发流程，将整个项目分解为多个可管理的里程碑。开发将遵循增量实现、视觉优先的原则。

---

### 里程碑 1: 核心渲染引擎与视觉基础 (Foundation & Core Rendering)

**目标**: 搭建项目基础，实现核心的视觉风格和渲染能力。

1.  **项目初始化**:
    *   使用 Vue.js (`@vue/cli`) 创建项目框架。
    *   设置 Git 版本控制，并创建 `.gitignore` 文件以排除 `node_modules`、构建产物等。
2.  **渲染引擎**:
    *   选择并集成渲染技术（优先考虑 HTML5 Canvas 或 WebGL 以获得高性能）。
    *   封装基础的几何图形绘制函数（如：线条、矩形、棱形）。
3.  **视觉风格实现**:
    *   实现核心的辉光（Glow）和光污染（Light Pollution）着色器（Shader）或效果。
    *   建立灰白为主的调色板，并应用到基础渲染中。
4.  **基础游戏循环**:
    *   创建一个简单的游戏循环（Game Loop），能够持续刷新渲染画面。

---

### 里程碑 2: 核心游戏玩法实现 (Core Gameplay Mechanics)

**目标**: 实现音乐游戏的核心玩法和判定机制。

1.  **谱面系统**:
    *   设计并实现谱面文件格式（如 JSON），用于定义音符、判定线移动等。
    *   开发谱面加载和解析模块。
2.  **音符系统**:
    *   实现四种基本音符（短键、滑键、长键、黄条）的下落逻辑与渲染。
    *   实现音符与音乐的同步机制。
3.  **判定线系统**:
    *   实现判定线的动态移动，使其能够根据谱面数据进行平移、旋转等。
4.  **判定与计分**:
    *   实现音符的打击判定逻辑 (Perfect, Good, Bad, Miss)。
    *   开发计分系统和连击（Combo）系统。
    *   添加基础的点击音效和视觉特效。

---

### 里程碑 3: UI/UX 与主菜单 (UI/UX & Menus)

**目标**: 构建完整的用户界面和交互流程。

1.  **主菜单**:
    *   实现具有科技感的动态主菜单（如浮动结晶、微动画背景）。
2.  **谱面选择**:
    *   开发图集式的谱面选择界面。
3.  **游戏内 UI**:
    *   在游戏场景中添加分数、进度条、暂停按钮等 UI 元素。
4.  **设置菜单**:
    *   实现设置界面，允许玩家调整音量、下落速度、背景亮度、延迟等。

---

### 里程碑 4: 自制谱面工坊 (Chart Editor/Workshop)

**目标**: 为玩家和谱师提供强大的自制谱面工具。

1.  **工坊基础架构**:
    *   设计工坊的用户界面，主要面向 PC 端操作。
2.  **谱面编辑功能**:
    *   实现时间轴，并允许在时间轴上放置和编辑各种类型的音符。
    *   提供判定线运动轨迹的可视化编辑工具。
3.  **资源管理**:
    *   允许用户导入音频、谱面封面等。
4.  **导入与导出**:
    *   实现谱面的保存与导出功能，使其能被游戏本体加载。

---

### 里程碑 5: 平台适配与发布 (Platform Adaptation & Release)

**目标**: 优化并打包应用，准备发布。

1.  **安卓平台适配**:
    *   使用 Capacitor 或类似框架将 Web 应用打包为安卓 APK。
    *   优化移动端触摸操作和性能。
2.  **性能优化**:
    *   对渲染和游戏逻辑进行性能分析和优化，确保在目标设备上流畅运行。
3.  **收尾与润色**:
    *   添加更多的音效和视觉特效。
    *   进行全面的 Bug 测试与修复。
    *   准备应用商店上架所需资料。
