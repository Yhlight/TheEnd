## 终末
终末是一款音乐节奏游戏，它与Phigros，Phira，Phigrim等音游高度相似  
即终末也是一款下落式音游  
与这些音游类似，TheEnd的判定线将参与乐谱的演出，以此提供更好的演出效果  

### 编程语言
TheEnd使用HTML + CSS + JavaScript等技术栈进行开发，如果有需求，可以使用C++ + WASM辅助开发  
TheEnd极其注重画面效果，其要素多以方块，线条，辉光的形式堆叠而成，例如点击效果，会焕发出由多个方块组成的动画，TheEnd将使用大量堆叠的线条和方形物件构成画面的点点滴滴，然后使用辉光来点缀  

### 要求
1. TheEnd使用HTML + CSS + JavaScript + Vue.js等技术栈进行开发，如果有需求，可以使用C++ + WASM辅助开发  
2. 需要使用版本隔离隔离那些源代码之外的文件，防止推送到远程仓库  
3. 使用增量实现，注重画面的绘制  

### 画面
TheEnd极少依赖外部的图片，而是使用许多的方块，线条，棱形等方形的形状堆叠起整个世界  
然后通过适当的辉光，高光，光影，或光污染来增加画面的效果  
棱形，漂浮的结晶，微动画等效果会作为游戏的主菜单，或作为游戏的画面组成之一  
TheEnd面向二次元，或动漫爱好者，所以对画面的演出和效果很有追求  
颜色偏淡(偏白色或灰色)或偏深(深红或黑色)，但是不能靠中间  

音游铺会像图集一样整齐排放，开发者可以通过手机单击或鼠标左键点击进入铺面的游玩  
左上为设置，可以暂停，中间是进度，右边是分数  
所有的内容都多以方形为主，通过键与判定线绘制画面效果  

### 按钮
TheEnd延续Phigros等音游的的设置，分为短键，滑键，长键，黄条(只需要接住就可以了)  

### 设置
可以调整音量的大小，键位的大小，下落的速度，背景的亮度，音效的延迟等  

### 自制铺
TheEnd会提供一个工坊，供给开发者自制铺面，包括但不限于游戏中出现的所有组件，铺面的封面，音频等等重要信息，以及组件如何运行，如何在某段时间刻中运动  

### 安卓
TheEnd主要提供于安卓方面，电脑端更适合用来制作铺面
