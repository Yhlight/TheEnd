# TheEnd 项目开发路线图

## 阶段一：项目初始化 (预计完成时间：2025-11-16)

- [ ] 搭建 Vue.js 项目
- [ ] 配置开发环境 (Webpack/Vite, Babel, ESLint)
- [ ] 初始化 Git 仓库
- [ ] 创建 `.gitignore` 文件，隔离 `node_modules`, `dist` 等目录

## 阶段二：核心玩法开发 (预计完成时间：2025-12-14)

- [ ] 实现游戏主循环
- [ ] Note 数据结构设计
- [ ] 实现四种基本 Note 类型 (短键, 滑键, 长键, 黄条)
- [ ] Note 下落与移动逻辑
- [ ] 判定逻辑与计分系统
- [ ] 可移动判定线的实现
- [ ] 谱面解析器开发

## 阶段三：视觉与音效 (预计完成时间：2026-01-11)

- [ ] 使用 HTML5 Canvas 或 WebGL 进行画面渲染
- [ ] 实现基础的点击特效和 Note 消失动画
- [ ] 根据 `TheEnd.md` 的描述，设计并实现以方块和线条为主的视觉风格
- [ ] 音频管理器开发，处理背景音乐和音效的加载与播放
- [ ] 实现音频延迟校准功能

## 阶段四：UI/UX 开发 (预计完成时间：2026-02-08)

- [ ] 主菜单界面设计与实现
- [ ] 曲目选择界面，支持谱面预览和难度选择
- [ ] 游戏内 UI (暂停按钮, 进度条, 分数显示)
- [ ] 设置界面 (音量, 下落速度, 背景亮度等)

## 阶段五：谱面编辑器 (预计完成时间：2026-03-22)

- [ ] 编辑器界面设计与实现
- [ ] 谱面可视化编辑功能
- [ ] 支持音频导入、封面设置、谱面信息编辑
- [ ] Note 的放置、拖拽、删除、修改
- [ ] 判定线移动轨迹的可视化编辑
- [ ] 谱面导出与分享功能

## 阶段六：安卓端构建与发布 (预计完成时间：2026-04-12)

- [ ] 使用 Cordova 或 Capacitor 将网页应用打包成安卓 APK
- [ ] 针对移动端进行性能优化和触摸操作适配
- [ ] 在不同分辨率的安卓设备上进行测试
- [ ] 准备发布材料，上架应用商店

## 技术栈

- **前端框架:** Vue.js
- **渲染技术:** HTML5 Canvas / WebGL (可能使用 Pixi.js 或 Three.js)
- **安卓打包:** Apache Cordova / Capacitor
- **版本控制:** Git
- **辅助开发:** C++ / WASM (按需)
